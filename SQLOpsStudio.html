SQL Operations Studio Localization Status



SQL Operations Studio (preview) is a free tool that runs on Windows, macOS, and Linux, for managing SQL Server, Azure SQL Database, and Azure SQL Data Warehouse; wherever they're running.

Contribution is opened for 10 languages: French, Italian, German, Spanish, Simplified Chinese, Traditional Chinese, Japanese, Korean, Russian, Brazilian Portuguese.

Here is where you can sign up to contribute to the translation of the Software/User Interface, while here are the Docs.

More information on
  • how to contribute to software localization
  • how to contribute to Docs
  • If you need more context during translations to see how the strings are displayed in the actual product, you can check the screenshots in the docs "How-to guides"

Requesting a new language

  • We can also extend the languages if there is strong demand from the Community
  • To do so, you can create a new request by opening a new Issue in the SQL Operations Studio github project. We suggest you put '[loc]Request to add xxx language to SQL Operations Studio' in the title.
  • Once we receive at least 3 requests, the language will be sent to the Product Team for approval. If approved, the language will be enabled typically within a week.
  • Until 100% translation is reached for a language, we will not submit those strings to SQL Operations Studio source repo.
