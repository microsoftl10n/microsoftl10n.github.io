Developer Tools Localization Project Status



Welcome to the Developer Tools Localization Project page. This page includes the description and localization status of each Open Source projects from Developer Tools.

Developer Tools Project List

    Library Manager

    Install and consume 3rd-party client-side libraries with ease.

    More info can be found at Library Manager on GitHub

    Managed Extensibility Framework (MEF)

    The Managed Extensibility Framework or MEF is a library for creating lightweight, extensible applications. It allows application developers to discover and use extensions with no configuration required. It also lets extension developers easily encapsulate code and avoid fragile hard dependencies. MEF not only allows extensions to be reused within applications, but across applications as well.

    More info can be found at Managed Extensibility Framework on GitHub

    Microsoft Test Framework (MSTest V2)

    Welcome to the "MSTest V2" repository, the evolution of the Microsoft Test Framework and Adapter. "MSTest V2" is currently in use in a variety of scenarios including:
    • in the relevant in-box unit test project templates (Visual Studio 2017 Preview 4 onwards)
    • in the Create Unit Tests wizard (Visual Studio 2017 Preview 4 onwards)
    • in the Create IntelliTest wizard (Visual Studio 2017 Preview 4 onwards)
    This is a fully supported, open source and cross-platform implementation of the MSTest test framework with which to write tests targeting .NET Framework, .NET Core and ASP.NET Core on Windows, Linux, and Mac. You can read more about MSTest V2 here. For API documentation refer here.

    More info can be found at Microsoft Test Framework (MSTest V2) on GitHub

    slow-cheetah

    Transformations for XML files (such as app.config) and JSON files.
    Includes two primary components:
    • NuGet package that adds an msbuild task to perform transforms on build.
    • Visual Studio extension for generating and previewing transforms.
      • This project has adopted the Microsoft Open Source Code of Conduct. For more information see the Code of Conduct FAQ or contact opencode@microsoft.com with any additional questions or comments.

    More info can be found at slow-cheetah on GitHub

    StreamJsonRpc

    StreamJsonRpc is a cross-platform, .NET portable library that implements the JSON-RPC wire protocol. It works over Stream or WebSocket independent of the underlying transport.

    More info can be found at StreamJsonRpc on GitHub

    Symreader Converter

    Converts between Windows PDB and Portable PDB formats.

    More info can be found at Microsoft.DiaSymReader.Converter on GitHub

    VS Threading

    The Microsoft.VisualStudio.Threading is a xplat library that provides many threading and synchronization primitives used in Visual Studio and other applications.

    More info can be found at Microsoft.VisualStudio.Threading on GitHub

    VSTS Tasks Tool Installer SDK

    Tool installer tasks SDK for Visual Studio Team Services build and deployment.

    More info can be found at VSTS Tasks Tool Installer SDK on GitHub

    Microsoft.VisualStudio.Validation (VSValidation)

    This project is available as the Microsoft.VisualStudio.Validation NuGet package.

    More info can be found at Microsoft.VisualStudio.Validation on GitHub
